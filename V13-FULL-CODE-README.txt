V13-FULL-CODE 덮어쓰기용 코드 ZIP

적용 방법
1. 이 ZIP을 압축 해제합니다.
2. 압축 해제된 내부 파일과 폴더를 기존 홈페이지 최상위 폴더에 복사합니다.
3. 같은 이름의 파일이 나오면 '덮어쓰기'를 선택합니다.
4. 기존 홈페이지 폴더를 먼저 삭제하지 마십시오.

포함 범위
- HTML, CSS, XML, TXT, JSON, Markdown 및 배포 설정 파일
- 원래 홈페이지와 동일한 상대 폴더 구조

제외 범위
- JPG, JPEG, PNG, WEBP, GIF, SVG
- MP4, MOV, AVI, WEBM
- PDF 및 기타 미디어

따라서 덮어쓰기 방식으로 복사하면 ZIP에 없는 기존 사진과 영상은 삭제되지 않습니다.
